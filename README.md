# 緑ヶ丘調剤薬局 情報共有ポータル

薬局スタッフ向けの情報共有・業務管理ポータルサイトです。

## 機能

- **ダッシュボード**: シフト・お知らせ・薬剤情報の概要表示、カスタマイズ対応
- **お知らせ掲示板**: 業務連絡・研修情報の投稿・管理
- **シフト管理**: カレンダー形式でのシフト表示・編集
- **薬剤情報管理**: 薬剤データベースの検索・登録・編集（テーブル/カード表示切替、高度なフィルタリング対応）
- **従業員管理**: スタッフ情報の管理
- **AIアシスタント**: Google Gemini APIを活用した薬局業務支援チャット

## 技術スタック

- HTML5 / CSS3 / JavaScript (Vanilla)
- Font Awesome 6.4
- Google Fonts (Noto Sans JP)
- localStorage によるデータ永続化

## 使い方

GitHub Pagesで公開されています。ブラウザからアクセスするだけで利用可能です。
データはブラウザのlocalStorageに保存されます。
