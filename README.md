# 緑ヶ丘調剤薬局 - 情報共有ポータル

緑ヶ丘調剤薬局のスタッフ向け情報共有ポータルサイトです。

## 機能

- **ダッシュボード**: 今日のシフト、最新のお知らせ、要注意薬剤、クイックアクセス（カスタマイズ可能）
- **お知らせ掲示板**: 業務連絡、研修情報などの投稿・管理
- **シフト管理**: カレンダー形式でのシフト表示・登録
- **薬剤情報管理**: 薬剤の登録・編集・検索・フィルタリング（テーブル/カード表示切替対応）
- **従業員管理**: スタッフ情報の管理
- **AIアシスタント**: Google Gemini APIを活用した薬局業務サポート

## 技術スタック

- HTML5 / CSS3 / JavaScript（バニラ）
- データ保存: localStorage（ブラウザ内保存）
- AI機能: Google Gemini API

## 使い方

GitHub Pagesで公開されています。ブラウザでアクセスするだけで利用可能です。
データはブラウザのlocalStorageに保存されるため、同じブラウザからアクセスすればデータが保持されます。
